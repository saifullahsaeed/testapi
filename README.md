# testapi

This is a test api for frontend developers to test there app behaviour 

Might be useful to test the app behaviour in the browser.

Migrate database before running the api.

Technologies used:

    -sqlite3
    -passport
    -express
    -body-parser
    -jsonwebtoken

Installation:

     npm install
     npm run dev

Migration:

    <code>
      npm run migrate
    </code>
